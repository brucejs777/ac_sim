# Three-Phase AC Simulator

Generate simulated 50/60 Hz 3-Phase voltage and current waveforms.

## Hardware
- Raspberry Pi Pico 2W
- Custom filter and step-up circuit and board.

## Building
1. Install Pico SDK
2. `mkdir build && cd build`
3. `cmake .. && make`
